Project Documentation:

Project Name: Random Password Generator using Python
Team Name   : Team Titans

Description of Project: In this project, the user has to select the password length and then 
click on the “Generate Password” button. It will show the generated password below. 
If the user clicks on the “Copy To Clipboard” button, then it will copy the password automatically.
